# Key Features

* Smart Ad Recommender: Recommends the top 3 advertising channels based on business type, goals, and location while strictly adhering to budget-based exclusion rules.

* Creative Ad Generator: Generates complete ad concepts, including scripts for social media reels, TV commercials, and poster designs using Gemini 2.0 Flash.

* Sales Dashboard: Visualizes budget allocation, projected ROI, and revenue growth over time using interactive charts.

* Secure Backend: Uses a Flask API to securely handle AI requests, keeping API keys hidden from the client-side browser.

# Tech Stack

* Frontend: HTML5, CSS3, JavaScript (ES6+), Chart.js.

* Backend: Python, Flask, Flask-CORS.

* AI Model: Google Gemini 2.0 Flash API.

* Environment Management: Python-dotenv for secure API key storage.

# Installation & Setup

* Python 3.8+ installed.

* A Google Gemini API Key
